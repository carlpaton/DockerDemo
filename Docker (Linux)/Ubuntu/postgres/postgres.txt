(1) Find the IP on the Linux VM
ip addr show

(2) Setup Postgres Docker Container on the Linux VM
sudo docker pull postgres
sudo docker container kill postgres_p6
sudo docker rm postgres_p6
sudo docker run --detach --name=postgres_p6 -p 192.168.231.129:62006:5432  --env="POSTGRES_PASSWORD=postgres" postgres
sudo docker start postgres_p6
sudo docker ps --all
